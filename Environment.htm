Windows Environment

To install Dragon on your Windows, all you need to do is download the installtion file from here and run the downloaded file.
You can also download the IDE (ZDragon) from here.

Linux Environment

Linux, you can install Dragon by downloading the zipped installer from the download URLs above. Then open your terminal and type the following:

  • First, install the unzip package (skip this if you already have it installed).
    $sudo apt install unzip
  • Unzip the downloaded zip file.
    unzip Dragon-linux1.9.7.zip
  • Navigate to the directory you unzipped the file.
    cd Dragon-linux1.9.7
  • Change the file permission and make install.sh executable.
    chmod +x install.sh
  • Then run your install.sh.
    ./install.sh

To install ZDragon IDE on Linux, all you to do is to open you terminal and do this:

wget https://dragon-lang.org/installZDragon1.6

Mac OS

  • Download the zipped Mac OS installer.
  • Unzip the downloaded file.
  • Double click this package file and follow the installation wizard instructions to install.

Linux AArch64 and Android (Userland or Termux)

For installing Dragon on Linux AArch64 and on Android (via Userland or Termux), you should simple run these commands in your terminal.

  • wget https://dragon-lang.org/installdragon-native
  • bash installdragon-native

To test if your installations are successful, run this command in your terminal or cmd

>> dragon

You will get an output similar to this:

Dragon version 1.9.7

Running Dragon

There are three different ways to run Dragon:−

1. Interactive Interpreter

You can run Dragon from Unix, DOS, or any other system that provides you a command-line interpreter or shell window.

Enter dragon -sh or dragon --shell in the command line.

Start coding right away in the interactive interpreter.

kayda@kali:-$ dragon -sh             // Unix/Linux
or
kayda@kali:-$ dragon --shell             // Unix/Linux
or
C:>dragon --sh           // Windows/DOS
or
C:\Users\kayda>dragon --sh           // Windows/DOS

2. Script from the Command-line

A Dragon script can be executed at the command line by invoking the interpreter on your application, as shown in the following example.

kayda@kali:-$ dragon -r hello.dgn          // Unix/Linux
or
kayda@kali:-dragon --run hello.dgn           // Unix/Linux
or 
C:>dragon -r hello.dgn         // Windows/DOS
or 
C:\Users\kayda>dragon --run hello.dgn         // Windows/DOS

Here is the list of all the available command line options:−

1. -sh,--shell:- Enter shell mode (use the terminal or command line to code).

2. -r,--run [filename].dgn:- Run Dragon script from given file

3. -s,--showtime:- Show elapsed time of parsing and executing of your Dragon script. (e.g. C:\Users\kayda>dragon -r hello.dgn -s)

Note − Be sure the file permission mode allows execution.

3. Integrated Development Environment (IDE)

You can run Dragon from a Graphical User Interface (GUI) environment as well, if you have a GUI application on your system that supports Dragon.

ZDragon − ZDragon is the first Windows and Linux IDE for Dragon.

Also, With Dragon extension for VSCode (one of the most popular code editor), you can use VSCode to code your Dragon program(s).

If you are not able to set up the environment properly, then you can take the help of your system admin.
You can also setup and make sure Dragon environment is properly set up and working perfectly fine.

You can also use the Dragon Programming environment online, so that you can execute all the available examples online. Check here for Dragon Online IDE and you can also use Repl.io which has support for Dragon.
Feel free to modify any example and execute it yourself.
